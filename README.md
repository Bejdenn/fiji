# Fiji

Fiji setups a fresh Fedora Workstation installation to act the way I need it to.
Maybe you can also benefit from some of the settings and applications here.

This repo started as a fork of [Omakub](https://github.com/basecamp/omakub) and as such has many scripts that are similar or equal to theirs.
Still, I have made some changes in Fiji that reflect my preferences.
